# Ultrasonic pi

testing ultrasonic sensor on raspberry pi with a second raspberry pi

## Getting started

```python
sudo apt-get install python-pip python-dev
sudo python3 -m pip install psutil
```

## Modules needed to run it

```python
psutil
flask
flask_cors
sseclient

greenlet
gevent
```
